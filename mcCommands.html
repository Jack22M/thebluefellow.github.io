About

This is part of my website for MCPE (although i mainly play JAVA) which show crazy (but easy!!!) things to do in MCPE with commands blocks. NOTE: THIS IS FOR WINDOWS 10/MCPE NOT JAVA OR CONSOLES

First you need to give your self a command block, heres how: /give @s command_block

Okay, now you have a command block you will need to place it

Click/tap (depending on your device) on Command Input. Command Input is where you will put all the commands.

Before we get to the crazy commands lets do some basic one, in commamd input type /weather clear (it clears the weather) /weather rain (makes it rain) /weather thunder (makes a thunderstorm). They were all the weather commands now lets do time commands /time set sunrise, /time set day, /time set noon, /time set sunset /time set night /time set midnight, all these commands you can tell what they do
